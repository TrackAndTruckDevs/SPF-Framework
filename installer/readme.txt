--- SPF FRAMEWORK ---

[ English ]

Installation:

There are two options for installing the framework:

1.  **Use the EXE file (recommended method):**
    Run `spf-framework.exe`. It will automatically find the game location and install all files into the correct folder.
    *   **Note:** The EXE file may request administrator privileges upon launch, but this is only needed to correctly identify the path to the game (as the installation path for the Steam version is likely stored in HKEY_LOCAL_MACHINE). Administrator rights might not be required for the actual file installation if the game folder is writable by a standard user.

2.  **Manual Installation:**
    If you do not trust installers, you can manually copy the entire contents of the `manualInstall` folder into the `<YOUR_GAME>\bin\win_x64\plugins\` directory.
    For example, if the game is installed at `C:\Program Files (x86)\Steam\steamapps\common\American Truck Simulator`, the final path would be `C:\Program Files (x86)\Steam\steamapps\common\American Truck Simulator\bin\win_x64\plugins\`.

[ 中文 (简体) ]

安装：

有两种安装框架的选项：

1.  **使用 EXE 文件（推荐方法）：**
    运行 `spf-framework.exe`。它将自动查找游戏位置并将所有文件安装到正确的文件夹中。
    *   **注意：** EXE 文件在启动时可能会请求管理员权限，但这仅用于正确定位游戏路径（因为 Steam 版本的安装路径可能存储在 HKEY_LOCAL_MACHINE 中）。如果游戏文件夹对于标准用户是可写的，则实际的文件安装可能不需要管理员权限。

2.  **手动安装：**
    如果您不信任安装程序，可以手动将 `manualInstall` 文件夹的全部内容复制到 `<您的游戏>\bin\win_x64\plugins\` 目录中。
    例如，如果游戏安装在 `C:\Program Files (x86)\Steam\steamapps\common\American Truck Simulator`，则最终路径为 `C:\Program Files (x86)\Steam\steamapps\common\American Truck Simulator\bin\win_x64\plugins\`。
